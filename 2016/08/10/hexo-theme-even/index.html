首页文章展现方式

首页默认会展示文章全文。也可以通过以下两种不同的形式在首页展示文章。

文章截断

在文章内容中添加 <!-- more --> 即可在首页显示时只显示其之上的内容。

并且会在 <!-- more --> 添加一个 Read more 的链接。

自定义文章简述

可在文章头信息中添加 description 字段，并填写自定义的文章简述。

就可以在首页显示文章简述而不显示文章内容。
