Emmet Hotkey

2016-08-24

Available actions

  1. Expand Abbreviation – Tab or Ctrl+E
  2. Interactive “Expand Abbreviation” — Ctrl+Alt+Enter
  3. Match Tag Pair Outward – ⌃D (Mac) / Ctrl+, (PC)
  4. Match Tag Pair Inward – ⌃J / Shift+Ctrl+0
  5. Go to Matching Pair – ⇧⌃T / Ctrl+Alt+J
  6. Wrap With Abbreviation — ⌃W / Shift+Ctrl+G
  7. Go to Edit Point — Ctrl+Alt+→ or Ctrl+Alt+←
  8. Select Item – ⇧⌘. or ⇧⌘, / Shift+Ctrl+. or Shift+Ctrl+,
  9. Toggle Comment — ⇧⌥/ / Shift+Ctrl+/
  10. Split/Join Tag — ⇧⌘' / Shift+Ctrl+`
  11. Remove Tag – ⌘' / Shift+Ctrl+;
  12. Update Image Size — ⇧⌃I / Ctrl+U
  13. Evaluate Math Expression — ⇧⌘Y / Shift+Ctrl+Y
  14. Reflect CSS Value – ⇧⌘R / Shift+Ctrl+R
  15. Encode/Decode Image to data:URL – ⇧⌃D / Ctrl+'
  16. Rename Tag – ⇧⌘K / Shift+Ctrl+'
  17. Merge Row – ⌘J

Increment/Decrement Number actions:

  • Increment by 1: Ctrl+↑
  • Decrement by 1: Ctrl+↓
  • Increment by 0.1: Alt+↑
  • Decrement by 0.1: Alt+↓
  • Increment by 10: ⌥⌘↑ / Shift+Alt+↑
  • Decrement by 10: ⌥⌘↓ / Shift+Alt+↓
